How to Deploy These Scripts to Google Sheets

  • Open a Google Sheet and go to Extensions → Apps Script.
  • Copy each .gs file from the relevant docs/ or doc/ folder into the editor.
  • For each system, run the setup/genesis function first to initialize the sheet structure.
  • Or use Clasp: clasp push from the project directory.

Ma Assayer — Gold Universe Purity Engine v4.3.0 ma_assayer/docs/

"Testing the purity of predictions — separating Gold from Charcoal"

Consumes raw satellite data, runs purity analysis, and produces two output sheets (ASSAYER_EDGES and ASSAYER_LEAGUE_PURITY) that become the official contract for the Mothership.

⬡ Platinum ≥85% Au Gold ≥72% Ag Silver ≥62% Cu Bronze ≥55% ite Rock ≥50% 🜃 Charcoal <50%
M1

Output Writers

Writes the official purity contract to ASSAYER_EDGES and ASSAYER_LEAGUE_PURITY sheets.

Output_Writers.gs
M2

Main Orchestrator

Controller that coordinates the full assay pipeline end-to-end when "Run Full Assay" is triggered.

Main_Orchestrator.gs
M3

Flagger

Applies purity flags back to source sheets, marking each record with its grade (Gold, Silver, Charcoal, etc.).

Flagger_FlagsSourceSheets.gs
M4

Discovery Edge Engine

Discovers and validates statistical edges across leagues, quarters, tiers, and confidence buckets. Handles type-segmented totals.

Discovery_Edge.gs
M5

Config & Constants

Single source of truth for all sheet names, column schemas, spread/line/confidence buckets, grade thresholds, and statistical parameters.

Config_ConfigurationAndConstants.gs
M6

ColResolver — Fuzzy Matching

Case-insensitive fuzzy column matching using Levenshtein distance and word-level similarity. Maps any header variant to its canonical name.

ColResolver_ColumnMatching.gs
M7

Parser

Parses raw data rows from Side and Totals sheets. Extracts dates, leagues, picks, results, confidence, EV, and tier information.

Parser_DataParsing.gs
M8

Stats Calculations

Wilson confidence intervals, shrinkage estimators, Brier score calibration, lift scoring, and reliability gating.

Stats_StatsCalculations.gs
M9

Utility Functions

Shared helpers: date normalisation, safe number parsing, sheet existence checks, and data sanitisation utilities.

Utils_UtilityFunctions.gs
M10

Logging System

Structured, levelled logging (debug/info/warn/error/success) with module namespacing and optional sheet output.

Log_LoggingSystem.gs
Config

Config Ledger Reader

Reads the Config_Ledger sheet to track config stamp versions, detect drift between satellite configurations, and validate stamp purity.

ConfigLedger_Reader.gs

Ma Golide Mothership — The Parent of the Gold Universe Central Brain ma_golide_mothership/doc/

"The one who sees all satellites, consults the Assayer, and builds the Accas"

Reads upcoming games from all satellite sheets, pulls the latest purity report from the Assayer, then produces Accas, Risky Accas, Portfolios, and Bet Slips.

Core

Intelligence Core (MIC)

The learning engine. Maintains historical archives, segment stats, Bayesian win-rate tracking, shadow backtest, and auto-tuner. Decides BET / CAUTION / BLOCK per segment.

Mothership_ Intelligence_Core.gs
Core

Acca Engine

Assembles standard accumulators from satellite data filtered through Assayer purity grades. Applies confidence tiers and EV thresholds.

Mothership_AccaEngine.gs
Bridge

Assayer Bridge

Multi-satellite bridge. Connects the Mothership to all registered satellite spreadsheets and pulls the Assayer's ASSAYER_EDGES purity contract.

Mothership_AssayerBridge.gs
Setup

Genesis

One-time bootstrapper that creates all Mothership sheet schemas (Archives, Performance Log, Segment Stats, Insights, Policy Overrides, etc.).

Mothership_Genesis.gs
AI

Hive Mind

Aggregate intelligence layer. Synthesises insights across all satellites, identifies system-wide patterns, and generates strategic recommendations.

Mothership_HiveMind.gs
UI

Menu

Google Sheets custom menu builder. Exposes all Mothership operations (sync, build accas, run MIC, tune, export) as one-click menu items.

Mothership_Menu.gs
Risk

Risky Acca Builder

Constructs higher-risk accumulator portfolios using expanded confidence ranges. Includes variance budgeting and kelly-fraction sizing.

Mothership_RiskyAccaBuilder.gs
Risk

Risky Analyzer

Post-hoc analysis of risky acca performance. Tracks hit rates, ROI contributions, and leg-level failure attribution.

Mothership_RiskyAnalyzer.gs
Analytics

Performance Analyzer

Full performance reporting pipeline. Computes win rates (lifetime, L30, L10), ROI, trend direction, and segment-level health scores.

Performance_Analyzer.gs
Audit

Systems Audit

End-to-end integrity check across all Mothership sheets. Validates schema compliance, detects orphaned records, and flags data drift.

Systems Audit.gs
Config

Config Ledger Mothership

Reads Config_Ledger from the Assayer sheet. Segments rows by config stamp, detects win-rate drift across versions, and gates accas via isSafeToAcca().

ConfigLedger_Mothership.gs
